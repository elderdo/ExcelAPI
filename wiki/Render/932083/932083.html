What is in it?

This package contains all the sample projects of C#. All C#, VB.NET solutions are offered in Microsoft Visual Studio 2010 and 2012 for users respectively.

We use Free Spire.XLS for .NET to operate the Excel files generated by NPOI. Free Spire.XLS supports .NET2.0, .NET 3.5, .NET 4.0 and .NET 4.5. The sample in this download package files are in .NET 4.0.

Where to download and how to install?

Here you can download the NPOIExcel. Spire.XLS dll is independent with sample projects.

How to run the Demos?

Once you download the page and unzip it, both "Converts" and "Demo" list the samples to you. "Converts" shows you how to convert Excel into PDF, HTML, TXT, XML and Image. Demo shows you how to copy worksheet in excel, create pivot table, generate excel charts, import data from Excel data table, merge excel files, remove hyperlinks from excel and resize and move excel worksheets.
